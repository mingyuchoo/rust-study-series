## 배경 및 목적
현재 SPA 와 help 페이지는 한국어만 지원한다. 외국 사용자도 사용할 수 있도록 두 페이지 모두 ko/en 전환 가능한 UI를 추가한다.

## 요구사항
- **FR-1**: 헤더 우측에 `[한] [EN]` 2-버튼 토글. 선택된 언어는 `localStorage.lang`에 저장되어 새로고침 시 복원.
- **FR-2**: index.html의 헤더·네비탭·각 패널 레이블/버튼/힌트 문자열을 `data-i18n="key"` 마킹 + JS 사전(`const I18N={ko:{...},en:{...}}`)으로 전환.
- **FR-3**: help.html은 본문이 장문이므로 `<main class="lang-ko">` + `<main class="lang-en">` 두 블록을 제공하고 선택된 언어만 표시.

## 성공 기준
- `[EN]` 클릭 시 모든 SPA 요소가 영어로 즉시 전환
- 새로고침 후에도 마지막 선택이 유지됨
- help 페이지에서도 동일하게 동작
- 테스트: i18n 사전/토글 버튼/`data-i18n` 속성/두 언어 블록 존재 검증
