## 배경 및 목적
현재 웹 SPA 와 사용안내 페이지는 `-apple-system`/`Segoe UI` 기본 시스템 폰트를 사용한다. 한글·영문이 섞인 UI 에서 일관된 타이포그래피를 위해 IBM 의 오픈 소스 폰트 패밀리(IBM Plex 시리즈)로 통일한다.

## 요구사항

### 기능 요구사항
- **FR-1**: `body` 와 UI 요소의 `font-family` 에 `"IBM Plex Sans KR", "IBM Plex Sans", -apple-system, "Segoe UI", sans-serif` 스택 적용. 한글은 Plex Sans KR, 영문은 Plex Sans 가 자동 선택된다.
- **FR-2**: `code, pre, textarea` 의 `font-family` 를 `"IBM Plex Mono", "SF Mono", Menlo, monospace` 로 교체.
- **FR-3**: 두 페이지(`index.html`, `help.html`) 모두 `<head>` 에 Google Fonts `<link>` 삽입:
  ```html
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link href="https://fonts.googleapis.com/css2?family=IBM+Plex+Mono:wght@400;600&family=IBM+Plex+Sans+KR:wght@400;600&family=IBM+Plex+Sans:wght@400;600&display=swap" rel="stylesheet">
  ```

### 비기능 요구사항
- **NFR-1**: 폰트 로딩 실패(오프라인) 시 시스템 폰트로 자연스럽게 폴백.
- **NFR-2**: `display=swap` 으로 FOUT 허용해 첫 렌더 지연 최소화.

## 성공 기준
- 브라우저/WebView 에서 렌더링 시 IBM Plex 글꼴이 적용
- 단위 테스트: 임베드된 HTML 본문에 `IBM Plex Sans KR`, `IBM Plex Mono`, `fonts.googleapis.com` 키워드 포함

## 범위 외
- 폰트 파일 로컬 번들링 (오프라인 전용 배포)
- 가변 폰트 weight 축 전체 노출 (300/700 등)
- 글자 간격/행간 대규모 재조정
