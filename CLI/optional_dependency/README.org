* Optional Dependency
** How to use
#+name: Cargo.toml
#+BEGIN_SRC toml
  [dependencies]
  some_crate = { version = "1.0", optional = true }
  [features]
  extra_functionality = ["some_crate"]
#+END_SRC
** How to build
1. Not activate JSON feature
#+BEGIN_SRC shell
  cargo run
#+END_SRC
2. Activate JSON feature
#+BEGIN_SRC shell
  cargo run --features json
#+END_SRC
