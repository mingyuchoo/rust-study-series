## 배경 및 목적
개발자와 글로벌 팀이 여러 타임존의 현재 시간을 터미널에서 빠르게 확인할 수 있는 CLI 도구가 필요하다. 웹 브라우저를 열지 않고 터미널에서 즉시 시간을 확인할 수 있어야 한다.

## 대상 사용자
- 글로벌 팀과 협업하는 개발자
- 다양한 타임존의 시간을 자주 확인해야 하는 사용자
- 터미널 기반 워크플로우를 선호하는 사용자

## 요구사항
### 기능 요구사항 (Functional)
- FR-1: 저장된 도시 목록의 현재 시간을 테이블 형식으로 표시한다. 도시명, 타임존, 현재 시간, UTC 오프셋을 포함한다.
- FR-2: 도시명과 IANA 타임존 문자열을 입력받아 설정에 추가한다. 중복 도시명은 에러를 반환한다. 잘못된 타임존은 에러를 반환한다.
- FR-3: 도시명을 입력받아 설정에서 삭제한다. 존재하지 않는 도시명은 에러를 반환한다.
- FR-4: 저장된 도시 목록을 도시명과 타임존 형식으로 출력한다.

### 비기능 요구사항 (Non-Functional)
- NFR-1: 설정은 JSON 파일로 OS별 표준 설정 디렉토리에 저장한다.
- NFR-2: 첫 실행 시 설정 파일이 없으면 빈 설정으로 동작한다 (에러 아님).
- NFR-3: --config 옵션으로 설정 파일 경로를 지정할 수 있다.

## 성공 기준
- 모든 CLI 명령어가 정상 동작한다.
- 잘못된 입력에 대해 명확한 에러 메시지를 출력한다.
- 설정 파일의 읽기/쓰기가 안정적이다.

## 범위 외 (Out of Scope)
- TUI (대화형 인터페이스)
- 실시간 시간 업데이트 (watch 모드)
- 타임존 자동 완성/검색

## 제약 조건
- Rust 2024 edition 사용
- 크로스 플랫폼 지원 (Windows, macOS, Linux)
