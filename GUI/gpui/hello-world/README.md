# hello-world

Create GPUI App으로 생성된 기본 gpui 예제 프로젝트입니다.

## 주요 의존성

- **gpui 0.2.2**: GPU 가속 UI 프레임워크
- **gpui-component 0.3.1**: gpui 컴포넌트 라이브러리
- **anyhow 1.0**: 에러 처리

## 실행 방법

- Rust가 설치되어 있어야 합니다 - [Rustup](https://rustup.rs/)
- `cargo run` 명령으로 실행
